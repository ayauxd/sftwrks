# Softworks Trading Company - Image Generation Prompts

## Visual Design System Reference

### Brand Color Palette
| Color Name | Hex Code | Usage |
|------------|----------|-------|
| Deep Navy | `#0A1628` | Primary background |
| Steel Blue | `#1E3A5F` | Dimensional elements, mid-tones |
| Slate Gray | `#2C3E50` | Secondary surfaces, shadows |
| Electric Cyan | `#00D4FF` | Glows, data flows, illumination, accents |
| Soft Cyan | `#4ECDC4` | Secondary highlights |
| Circuit Gold | `#FFD700` | Accent highlights (sparingly) |
| Pure White | `#FFFFFF` | Text, high contrast elements |
| Warm Paper | `#F5F5F4` | Light mode backgrounds (reference) |

### Visual Language Guidelines
- **Style**: Paper-craft / 3D diorama aesthetic with depth, layers, and dimensional shadows
- **Figures**: Silhouette humans, no detailed facial features, professional postures
- **Textures**: Circuit board patterns, binary code streams, architectural blueprints
- **Core Metaphor**: Bridge connecting chaos (tangled wires/uncertainty) to structured city (clarity/outcomes)
- **Lighting**: Dramatic spotlights, cyan data glows, flashlight beams revealing paths
- **Mood**: Professional, architectural, infrastructural, purposeful

## 1. Hero Image - "The Bridge to Clarity"

### Primary Prompt
```
A stunning paper-craft 3D diorama scene depicting a lone silhouette figure standing on an illuminated bridge, holding a glowing cyan flashlight that casts a beam forward, revealing an organized, geometric city skyline ahead. The figure is positioned at the midpoint of an elegant suspension bridge constructed from layered paper with visible depth and shadows.

Behind the figure: chaotic tangles of wires, fragmented circuit boards, broken data streams represented as torn paper ribbons in disarray, representing confusion and complexity. The background chaos uses darker steel blue (#1E3A5F) and slate gray (#2C3E50) tones with sparking, disconnected cyan (#00D4FF) nodes.

Ahead of the figure: a pristine, illuminated paper cityscape with clean geometric buildings, organized data conduits glowing with electric cyan (#00D4FF), structured pathways, and ascending architectural towers. The city represents clarity, order, and successful outcomes.

The bridge itself features intricate circuit board patterns etched into its surface, with data flowing as luminous cyan streams along its cables. Binary code (0s and 1s) subtly textured into the bridge's paper surface. The entire scene is set against a deep navy (#0A1628) background that creates dramatic depth.

Lighting: The flashlight beam creates a cone of warm-white light revealing the path ahead, while ambient cyan glows emanate from the city and data conduits. Subtle paper texture and dimensional layering throughout, with visible cut-paper edges and shadow depth between layers.

Style: Highly detailed paper-craft diorama, architectural precision, professional and sophisticated mood, clean lines with organic chaos contrast.
```

### Negative Prompt
```
Realistic human faces, photographic people, cartoon style, flat illustration, bright saturated colors, red or orange dominant colors, messy composition, low quality, blurry, text overlays, watermarks, logos, childish appearance, fantasy creatures, magical elements, nature scenes, plants, animals, outdoors natural environment, lens flare, chromatic aberration, noise grain, overexposed, underexposed
```

### Specifications
- **Aspect Ratio**: 16:9 (landscape, hero banner)
- **Recommended Model**: Midjourney v6 (best for architectural paper-craft aesthetic)
- **Alternative Model**: DALL-E 3 (good for conceptual interpretation)
- **Resolution Target**: 2048 x 1152 minimum

## Model-Specific Notes

### Midjourney v6 (Primary Recommendation)
- Add `--style raw` for more literal interpretation
- Add `--stylize 50-100` for balanced creativity
- Use `--ar 16:9` or `--ar 3:4` as specified
- Consider `--chaos 10-20` for slight variation while maintaining coherence

**Example suffix**: `--ar 16:9 --style raw --stylize 75 --v 6`

### DALL-E 3
- Works well with detailed natural language descriptions
- May need multiple generations for optimal results
- Tends toward cleaner, less textured outputs
- Good for conceptual accuracy

### Gemini 3 Pro Image
- Strong with architectural and technical concepts
- Good color palette adherence
- May require simpler prompt structure
- Effective for blueprint and schematic imagery

## Post-Production Guidelines

1. **Color Correction**: Ensure deep navy backgrounds match `#0A1628` exactly
2. **Cyan Consistency**: All glowing elements should match `#00D4FF`
3. **Contrast Enhancement**: Increase shadows for dimensional depth
4. **Export Formats**:
   - WebP for web (primary)
   - PNG for transparency needs
   - JPEG for fallback
5. **Compression**: Optimize for web while maintaining quality (aim for under 500KB per image)
